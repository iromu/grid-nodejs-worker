## Install

```sh
$ npm install --save grid-nodejs-worker
```


## Usage

```js
var gridNodejsWorker = require('grid-nodejs-worker');

gridNodejsWorker('Rainbow');
```

```sh
$ npm install --global grid-nodejs-worker
$ grid-nodejs-worker --help
```

```sh
# creates a browser.js
$ npm run browser
```
